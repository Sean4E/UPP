Frequently Asked Questions

What Blender versions are supported?

Ultimate Palette Pro supports Blender 4.0 and newer. We actively test on the latest stable release.

Do I need any dependencies?

No! The addon is completely self-contained. The WebSocket server for web sync is built-in with no external libraries required.

How does the web sync work?

Start the server in Blender (Web Sync panel), then connect from the web app. Changes sync instantly in both directions over WebSocket on localhost.

Can I use this for commercial projects?

Absolutely! Once purchased, you can use Ultimate Palette Pro for any personal or commercial projects without restrictions.

Are updates included?
