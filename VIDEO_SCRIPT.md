## Visual Style Guide

### Colors
- Background: #0a0e17 (deep navy)
- Accent 1: #ff6b6b (coral red)
- Accent 2: #48dbfb (sky blue)
- Accent 3: #26de81 (mint green)
- Text: #ffffff

### Typography
- Headlines: Space Grotesk Bold
- Body: Space Grotesk Regular
- Code: JetBrains Mono

### Motion
- Easing: Cubic-bezier(0.4, 0, 0.2, 1)
- Duration: 300-500ms for UI elements
- Style: Smooth, professional, slightly playful

### Transitions
- Scene transitions: Cross-dissolve or wipe
- UI elements: Fade + slight scale (1.0 → 0.95 → 1.0)
- Color changes: Smooth interpolation

## Recording Checklist

### Before Recording
- [ ] Clean Blender install (or reset to defaults)
- [ ] Remove personal files from visible areas
- [ ] Set Blender theme to default dark
- [ ] Prepare demo scene with untextured objects
- [ ] Test WebSocket connection
- [ ] Close unnecessary applications

### During Recording
- [ ] Use slow, deliberate mouse movements
- [ ] Pause briefly on important UI elements
- [ ] Record at native resolution (1920x1080)
- [ ] Capture at 60fps for smooth motion
- [ ] Record multiple takes of each action

### After Recording
- [ ] Add cursor highlights/zoom where needed
- [ ] Add keyboard shortcut callouts
- [ ] Sync to music beats for polish
- [ ] Add captions/subtitles
- [ ] Export: H.264, High Quality, 1080p60

## Voiceover Notes

- Tone: Friendly, knowledgeable, slightly casual
- Pace: Moderate (not rushed)
- Emphasis words: "instantly", "magic", "one click", "free"
- Avoid: Overly salesy language, filler words

### If Self-Recording
- Use quiet room, away from echo
- Pop filter recommended
- Record at -12dB to -6dB
- Export as WAV or high-quality MP3
